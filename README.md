# Navy Todo App

濃紺と白を基調とした、落ち着いたデザインのプロジェクト管理型Todoアプリです。
シンプルながらもプロジェクトごとのタスク管理や、他ユーザーへのタスク割り当て機能などを備えています。

## 特徴

*   **Deep Navy Theme**: 集中力を高める濃紺 (`#0f172a`) と白のコントラスト。
*   **プロジェクト管理**: 「Personal」「Work」などのプロジェクトごとにタスクを整理。
*   **タスク割り当て**: 自分だけでなく、チームメンバー（Alice, Bobなど）にタスクを割り当て可能。
*   **モダンなUI**: CSS Modulesによるカスタムデザインで、プレミアムな操作感を実現。

## 技術スタック

*   **Framework**: Next.js 16 (App Router)
*   **Styling**: Vanilla CSS (CSS Modules)
*   **State**: React Local State (Mock Data)

## 始め方

開発サーバーを起動するには以下のコマンドを実行してください：

```bash
npm run dev
# or
yarn dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
